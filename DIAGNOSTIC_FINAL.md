### 2. Функции не определены
**Ошибки:** 
- `toggleReactionPicker is not defined`
- `editChatUserLabel is not defined`

**Причина:** Функции определены, но возможно выполняются до загрузки DOM  
**Решение:** Проверить порядок загрузки и добавить console.log для отладки

## Что нужно сделать

### Шаг 1: Очистить кеш браузера
1. Откройте DevTools (F12)
2. Нажмите Ctrl+Shift+Del
3. Очистите кеш
4. Перезагрузите с Ctrl+F5

### Шаг 2: Проверить загрузку скриптов
Откройте консоль (F12) и проверьте:
```javascript
console.log(typeof LocalUserLabels); // должно быть "function"
console.log(typeof window.userLabels); // должно быть "object"
console.log(typeof toggleReactionPicker); // должно быть "function"
console.log(typeof editChatUserLabel); // должно быть "function"
```

### Шаг 3: Проверить network tab
В DevTools → Network проверьте:
- `/static/local-labels.js` - должен быть 200, не 404
- `/static/encryption.js` - должен быть 200
- `/static/push-notifications.js` - должен быть 200
