## 📋 Функционал

### ✨ Основные возможности

- **Регистрация и аутентификация** - создание аккаунта и вход в систему

- **Система подписок** - подписка на других пользователей для быстрого доступа
- **Push-уведомления** 🆕 - реальные push-уведомления о новых сообщениях (работает на PythonAnywhere)
- **Удаление сообщений** - возможность удалить отправленные сообщения
- **Статус онлайн** - отображение статуса пользователей
- **Адаптивный дизайн** - работает на всех устройствах (десктоп, планшет, телефон)
- **Реакции на сообщения** 🆕 - эмодзи-реакции для быстрого выражения эмоций
- **Поиск по сообщениям** 🆕 - мгновенный поиск в истории чата
- **Темная тема** 🆕 - переключение между светлой и темной темой
- **Отправка файлов** - изображения, видео и документы с превью
- **Локальные метки** 🆕 - подписывайте пользователей своими именами (видны только вам)

### 🎨 Дизайн

- Современный градиентный дизайн
- Плавные анимации и переходы
- Интуитивный интерфейс
- Красивые карточки и элементы

## 🛠 Технологии

- **Backend**: Django 6.0
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite (для разработки), можно переключиться на PostgreSQL
- **Real-time**: Django Channels (WebSocket)
- **Push-уведомления**: Web Push API, pywebpush, VAPID
- **Загрузка файлов**: Pillow для обработки изображений

## 📦 Установка и запуск

### Требования

- Python 3.10+
- pip

### Шаги установки

1. Клонируйте репозиторий или распакуйте архив
2. Перейдите в директорию проекта:
```bash
cd shirokiy_messenger
```

3. Активируйте виртуальное окружение (уже создано):
```bash
# Windows
..\shirokiy_env\Scripts\activate

# Linux/Mac
source ../shirokiy_env/bin/activate
```

4. Установите зависимости (если еще не установлены):
```bash
pip install -r requirements.txt
```

5. Примените миграции (если еще не применены):
```bash
python manage.py migrate
```

6. Создайте суперпользователя для доступа к админ-панели:
```bash
python manage.py createsuperuser
```

7. Запустите сервер разработки:
```bash
python manage.py runserver
```

8. Откройте браузер и перейдите по адресу:
```
http://127.0.0.1:8000
```

## 🔐 Доступ к админ-панели

После создания суперпользователя:

```
http://127.0.0.1:8000/admin/
```

В админ-панели можно управлять:
- Пользователями
- Чатами
- Сообщениями
- Уведомлениями
- Подписками

## 📱 Использование

### Регистрация
1. Откройте главную страницу
2. Нажмите "Зарегистрироваться"
3. Заполните форму и подтвердите регистрацию

### Создание чата
1. Используйте поиск для нахождения пользователей
2. Нажмите "Написать" рядом с пользователем
3. Начните общение

### Создание группы
1. Нажмите "➕" на главной странице
2. Введите название группы
3. Загрузите аватар (необязательно)
4. Выберите участников
5. Нажмите "Создать группу"

### Редактирование профиля
1. Нажмите на иконку профиля (👤)
2. Нажмите "Редактировать"
3. Измените информацию
4. Загрузите новый аватар
5. Сохраните изменения

## 🔧 Настройка для разработки

### Изменение настроек

Откройте `shirokiy_messenger/settings.py` для изменения:

- `DEBUG` - режим отладки
- `ALLOWED_HOSTS` - разрешенные хосты
- `DATABASES` - настройки базы данных
- `LANGUAGE_CODE` - язык интерфейса
- `TIME_ZONE` - часовой пояс

### WebSocket (Real-time)

Для работы WebSocket в продакшене необходим сервер с поддержкой ASGI (например, Daphne или Uvicorn).

На бесплатном хостинге PythonAnywhere WebSocket не поддерживаются, поэтому используется fallback с автообновлением страницы.

## 🐛 Решение проблем

### Ошибка импорта модулей
```bash
pip install -r requirements.txt
```

### Ошибка миграции базы данных
```bash
python manage.py makemigrations
python manage.py migrate
```

### Не отображаются статические файлы
```bash
python manage.py collectstatic
```

### Не загружаются изображения
Проверьте права доступа к папке `media/`
