## 🐛 Проблема

**Описание:** Темная тема работала только на странице настроек, но не применялась к остальным страницам мессенджера.

**Причина:** Функция `loadTheme()` в base.html ссылалась на несуществующий элемент `themeToggle` (кнопка была удалена), из-за чего тема не загружалась при открытии других страниц.

---

## ✅ Решение

### 1. Исправлены функции в base.html

**Было:**
```javascript
function loadTheme() {
    const theme = localStorage.getItem('theme');
    const themeToggle = document.getElementById('themeToggle'); // ❌ Элемент не существует

    if (theme === 'dark') {
        document.body.classList.add('dark-theme');
        if (themeToggle) themeToggle.textContent = '☀️';
    }
}
```

**Стало:**
```javascript
function loadTheme() {
    const theme = localStorage.getItem('theme');

    if (theme === 'dark') {
        document.body.classList.add('dark-theme'); // ✅ Без зависимости от кнопки
    }
}
```

### 2. Добавлена глобальная функция setTheme

**Новая функция для страницы настроек:**
```javascript
window.setTheme = function(theme) {
    const body = document.body;

    if (theme === 'dark') {
        body.classList.add('dark-theme');
        localStorage.setItem('theme', 'dark');
    } else {
        body.classList.remove('dark-theme');
        localStorage.setItem('theme', 'light');
    }
};
```

### 3. Обновлена страница настроек

Теперь использует глобальную функцию `window.setTheme` вместо локальной:

```javascript
function setTheme(theme) {
    // Используем глобальную функцию из base.html
    if (window.setTheme) {
        window.setTheme(theme);
    }

    // Обновляем визуальное состояние карточек
    updateThemeCards(theme);
}
```

## 📊 Технические детали

### Порядок загрузки

1. Браузер загружает `base.html`
2. Выполняется `DOMContentLoaded`
3. Вызывается `loadTheme()`
4. Проверяется `localStorage.getItem('theme')`
5. Применяется класс `dark-theme` к `<body>`
6. CSS стили темной темы активируются

### localStorage

```javascript
// Структура данных
{
  "theme": "dark" // или "light"
}
```

### CSS классы

```css
body.dark-theme {
  /* Все стили темной темы */
  background: #0f172a;
  color: #e2e8f0;
  /* ... */
}
```
