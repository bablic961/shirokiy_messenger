# 🎉 Новые возможности Shirokiy Messenger

## Обновление от 19 июня 2026

Мы добавили несколько улучшений, чтобы сделать ваше общение ещё комфортнее и функциональнее!

---

## ✨ Что нового

### 1. 😊 Реакции на сообщения

Теперь вы можете выразить эмоции быстро с помощью реакций-эмодзи!

**Как использовать:**
- Наведите на любое сообщение в чате
- Нажмите кнопку **➕** под сообщением
- Выберите одну из 6 реакций: 👍 ❤️ 😂 😮 😢 🔥
- Ваша реакция появится под сообщением
- Нажмите на реакцию повторно, чтобы убрать её

**Особенности:**
- Можно добавить несколько разных реакций на одно сообщение
- Видно, кто поставил реакцию (при наведении)
- Реакции видны всем участникам чата

---

### 2. 🔍 Поиск по сообщениям

Больше не нужно листать историю! Найдите нужное сообщение за секунды.

**Как использовать:**
- Откройте чат
- Нажмите кнопку **🔍** в правом верхнем углу
- Введите текст для поиска (минимум 2 символа)
- Результаты появятся мгновенно
- Нажмите на результат, чтобы перейти к сообщению

**Особенности:**
- Поиск работает по содержимому текстовых сообщений
- Автоматическая подсветка найденного сообщения
- Показывает отправителя, время и превью сообщения

---

### 3. 🌙 Темная тема

Бережем ваши глаза! Переключайтесь между светлой и темной темой.

**Как использовать:**
- Нажмите на кнопку **🌙** в правом нижнем углу экрана
- Тема переключится на темную
- Иконка изменится на **☀️**
- Нажмите снова, чтобы вернуться к светлой теме

**Особенности:**
- Настройка сохраняется в браузере
- Применяется ко всем страницам
- Приятная темная палитра с синими акцентами
- Автоматическая загрузка сохраненной темы при входе

---

### 4. 📎 Улучшенное отображение файлов

Файлы теперь выглядят красивее и информативнее!

**Что улучшилось:**
- Разные иконки для разных типов файлов:
  - 📄 PDF документы
  - 📝 Word документы
  - 📊 Excel таблицы
  - 📦 Архивы (ZIP, RAR)
  - 📎 Остальные файлы
- Отображается полное имя файла
- Показывается размер файла
- Иконка скачивания ⬇️
- Ограничение размера изображений и видео для лучшего отображения

---

### 5. 🏷️ Локальные метки пользователей

Подписывайте пользователей своими метками, которые видны только вам!

**Как использовать:**
- Откройте профиль, чат или найдите пользователя в поиске
- Нажмите кнопку **🏷️** (метка)
- Введите свою подпись (например: "Иван из маркетинга", "Саша-сосед")
- Сохраните - метка будет отображаться везде рядом с именем

**Где доступно:**
- В профилях пользователей
- В шапке личных чатов
- В результатах поиска
- В списке чатов
- В групповых сообщениях

**Особенности:**
- Метки хранятся только на вашем устройстве (localStorage)
- Никто кроме вас их не видит
- Можно использовать эмодзи для категоризации
- Неограниченное количество меток
- Легко редактировать и удалять

**Примеры использования:**
- `user123` → **user123 (Иван Петров - HR)**
- `coolnick` → **coolnick (🔴 VIP клиент)**
- `designer` → **designer (Мария - дизайнер сайта)**

[Подробное руководство →](LOCAL_LABELS_GUIDE.md)

## 🎯 Технические улучшения

### Производительность
- Оптимизирована загрузка реакций с использованием `prefetch_related`
- Поиск работает с задержкой 300мс для снижения нагрузки
- Темная тема реализована через CSS без дополнительных запросов

### База данных
- Добавлена новая модель `MessageReaction`
- Уникальность реакций: один пользователь - одна реакция определенного типа на сообщение
- Автоматические миграции применены

### API
- Новый endpoint: `/api/add-reaction/<message_id>/` - добавление/удаление реакций
- Новый endpoint: `/api/search-messages/<chat_id>/` - поиск по сообщениям
- JSON-ответы для асинхронных операций

## 🚀 Как обновить

Если вы запускаете проект локально:

```bash
cd shirokiy_messenger
..\shirokiy_env\Scripts\activate  # Windows
python manage.py migrate
python manage.py runserver
```

Миграции будут применены автоматически!
