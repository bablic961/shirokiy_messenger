# 🏷️ Локальные метки пользователей

## Что это такое?

**Локальные метки** — это функция, которая позволяет вам добавлять собственные подписи (псевдонимы, заметки) к профилям других пользователей. Эти метки видны **только вам** на вашем устройстве и не отправляются на сервер.

## Зачем это нужно?

- 📝 Запомнить, кто есть кто (особенно если пользователи используют никнеймы)
- 💼 Добавить реальные имена к рабочим аккаунтам
- 🎯 Пометить важных контактов
- 📚 Организовать людей по категориям (например: "Коллега", "Друг из универа")

## Как использовать

### Добавить метку из профиля

1. Откройте профиль пользователя
2. Нажмите кнопку **🏷️ Метка**
3. Введите свою метку (например: "Иван из маркетинга")
4. Нажмите **💾 Сохранить**

Теперь рядом с именем пользователя будет отображаться ваша метка во всех местах мессенджера.

### Добавить метку из поиска

1. Перейдите в **🔍 Поиск пользователей**
2. Найдите нужного пользователя
3. Нажмите кнопку **🏷️** рядом с ним
4. Введите метку и сохраните

### Добавить метку из чата

1. Откройте личный чат
2. В шапке чата нажмите кнопку **🏷️** (рядом с именем собеседника)
3. Введите метку и сохраните

### Редактировать метку

- Просто откройте диалог редактирования снова
- Измените текст и нажмите **💾 Сохранить**

### Удалить метку

В диалоге редактирования метки:
- Нажмите кнопку **🗑️** (корзина)
- Или очистите поле и нажмите **💾 Сохранить**

## Где отображаются метки?

Метки появляются рядом с именами пользователей в:
- ✅ Профилях пользователей
- ✅ Шапке чата
- ✅ Результатах поиска
- ✅ Списке чатов
- ✅ Сообщениях в групповых чатах

## Формат отображения

После добавления метки имя пользователя отображается так:

**ИмяПользователя (Ваша метка)**

Например:
- `alex123` → **alex123 (Саша из спортзала)**
- `designer_pro` → **designer_pro (Мария - дизайнер)**

Метка выделена цветом для лучшей видимости.

## Важные особенности

### ✅ Преимущества

- **Приватность** — метки хранятся только на вашем устройстве в браузере
- **Никто не видит** — другие пользователи (включая того, кого вы пометили) не знают о ваших метках
- **Быстрый доступ** — можно редактировать из любого места
- **Неограниченное количество** — можете пометить сколько угодно пользователей
- **Любой текст** — используйте эмодзи, категории, полные имена

### ⚠️ Ограничения

- **Локальное хранилище** — метки хранятся в localStorage браузера
- **Не синхронизируются** — если вы зайдете с другого устройства или браузера, меток не будет
- **Очистка данных** — если очистите данные браузера, метки удалятся
- **Один браузер** — работает только в том браузере, где вы их создали

## Технические детали

### Где хранятся данные?

Метки сохраняются в `localStorage` браузера под ключом `userLabels` в формате JSON:

```json
{
  "user_id_1": "Иван Петров",
  "user_id_2": "Мария (Дизайнер)",
  "user_id_3": "Саша из спортзала"
}
```

### Безопасность

- ✅ Данные не отправляются на сервер
- ✅ Другие пользователи не имеют доступа
- ✅ Хранятся только в вашем браузере
- ✅ Автоматическое экранирование HTML для безопасности

### Резервное копирование

Если вы хотите сохранить свои метки:

1. Откройте консоль браузера (F12)
2. Выполните команду:
```javascript
console.log(localStorage.getItem('userLabels'));
```
3. Скопируйте и сохраните вывод

Для восстановления:
```javascript
localStorage.setItem('userLabels', 'ваши_сохраненные_данные');
```

## Часто задаваемые вопросы

**Q: Увидят ли другие мою метку?**  
A: Нет, метки хранятся только в вашем браузере и никому не видны.

**Q: Синхронизируются ли метки между устройствами?**  
A: Нет, метки привязаны к конкретному браузеру на конкретном устройстве.

**Q: Можно ли экспортировать метки?**  
A: Да, через консоль браузера (см. раздел "Резервное копирование").

**Q: Что произойдет, если я очищу кэш браузера?**  
A: Метки удалятся, так как они хранятся в localStorage.

**Q: Есть ли лимит на длину метки?**  
A: Технически нет, но рекомендуется держать метки короткими (до 50 символов) для удобства отображения.

**Q: Можно ли использовать эмодзи в метках?**  
A: Да! 😊 Эмодзи отлично работают и помогают визуально категоризировать контакты.

**Q: Влияют ли метки на поиск?**  
A: Нет, поиск работает только по оригинальным именам пользователей и email.
