# 📖 Руководство пользователя Shirokiy Messenger

## 🎯 Быстрый старт

### Запуск приложения

1. Откройте терминал в папке проекта
2. Активируйте виртуальное окружение:
   ```bash
   # Windows
   ..\shirokiy_env\Scripts\activate
   ```

3. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

4. Откройте браузер и перейдите по адресу:
   ```
   http://127.0.0.1:8000
   ```

### Учетные данные администратора

- **Логин**: admin
- **Пароль**: admin123
- **Админ-панель**: http://127.0.0.1:8000/admin/

## 🎨 Основные функции

### 1. Регистрация и вход

#### Регистрация нового пользователя
1. На главной странице нажмите "Зарегистрироваться"
2. Заполните форму:
   - Имя пользователя (уникальное)
   - Email
   - Пароль (минимум 8 символов)
   - Подтверждение пароля
3. Нажмите "Зарегистрироваться"
4. Вы автоматически войдете в систему

#### Вход в систему
1. На странице входа введите:
   - Имя пользователя
   - Пароль
2. Нажмите "Войти"

### 2. Главная страница

После входа вы увидите:
- **Левая панель**: список чатов и подписок
- **Поиск**: для поиска пользователей
- **Кнопки**: профиль, создание группы, выход

#### Вкладки
- **Чаты**: все ваши диалоги (личные и групповые)
- **Подписки**: пользователи, на которых вы подписаны

### 3. Поиск пользователей

1. В строке поиска введите имя пользователя или email
2. Нажмите Enter или кликните на иконку поиска
3. Выберите пользователя из результатов
4. Нажмите "Написать" для начала чата или "Профиль" для просмотра

### 4. Личные чаты

#### Создание чата
1. Найдите пользователя через поиск
2. Нажмите "Написать"
3. Вы автоматически перейдете в чат

#### Отправка сообщений
1. Введите текст в поле внизу страницы
2. Нажмите Enter или кнопку 📤
3. Сообщение появится в чате

#### Удаление сообщений
1. Наведите курсор на свое сообщение
2. Появится кнопка 🗑️
3. Нажмите и подтвердите удаление

### 5. Групповые чаты

#### Создание группы
1. На главной странице нажмите "➕"
2. Заполните информацию:
   - Название группы (обязательно)
   - Аватар группы (необязательно)
3. Выберите участников из списка
4. Нажмите "Создать группу"

#### Общение в группе
- Все сообщения видны всем участникам
- Отображается имя отправителя
- Можно удалять только свои сообщения

### 6. Профиль пользователя

#### Просмотр своего профиля
1. Нажмите на иконку 👤 в шапке
2. Вы увидите:
   - Аватар
   - Имя пользователя
   - Email
   - Биографию
   - Статистику (чаты, подписки, подписчики)

#### Редактирование профиля
1. В своем профиле нажмите "Редактировать"
2. Измените:
   - Имя пользователя
   - Email
   - Биографию (о себе)
   - Аватар (загрузите изображение)
3. Нажмите "Сохранить"

#### Просмотр чужого профиля
1. Найдите пользователя через поиск
2. Нажмите "Профиль"
3. Доступные действия:
   - Подписаться/Отписаться
   - Написать сообщение

### 7. Система подписок

#### Подписка на пользователя
1. Откройте профиль пользователя
2. Нажмите "Подписаться"
3. Пользователь появится во вкладке "Подписки"
4. Он получит уведомление о вашей подписке

#### Отписка
1. Откройте профиль пользователя
2. Нажмите "Отписаться"
3. Пользователь исчезнет из ваших подписок

#### Быстрый доступ
- Все ваши подписки отображаются во вкладке "Подписки"
- Нажмите на пользователя для начала чата
- Видны статусы онлайн/офлайн

### 8. Уведомления

1. Нажмите на иконку 🔔 (когда будет добавлена)
2. Увидите:
   - Новые сообщения
   - Новые подписки
   - Приглашения в группы
3. После просмотра уведомления помечаются как прочитанные

### 9. Статус онлайн

- **Зеленый индикатор**: пользователь онлайн
- **Нет индикатора**: пользователь офлайн
- Статус обновляется автоматически

### 10. Адаптивный дизайн

#### На мобильных устройствах
- Боковая панель занимает весь экран
- Чат открывается на полный экран
- Кнопка "← Назад" для возврата к списку чатов

#### На планшетах и десктопах
- Разделенный экран: чаты слева, сообщения справа
- Все функции доступны одновременно

## 🔧 Настройки и советы

### Загрузка аватарок
- Поддерживаемые форматы: JPG, PNG, GIF
- Рекомендуемый размер: 500x500px
- Максимальный размер файла: определяется настройками сервера

### Производительность
- Чаты автоматически обновляются каждые 5 секунд
- История сообщений загружается полностью
- Для улучшения производительности используйте Chrome или Firefox

### Безопасность
- Пароли хешируются и хранятся безопасно
- Используйте сложные пароли
- Не делитесь своими учетными данными

## ❓ Часто задаваемые вопросы

**Q: Как удалить свой аккаунт?**
A: Обратитесь к администратору через админ-панель

**Q: Можно ли восстановить удаленные сообщения?**
A: Нет, удаление необратимо

**Q: Сколько участников может быть в группе?**
A: Технически неограниченно, но рекомендуется до 100

**Q: Можно ли отправлять файлы?**
A: В текущей версии нет, но планируется в будущем

**Q: Как выйти из группы?**
A: Обратитесь к администратору группы или через админ-панель

**Q: Видят ли другие, когда я онлайн?**
A: Да, статус онлайн виден всем пользователям

**Q: Можно ли изменить цветовую тему?**
A: В текущей версии нет, но темная тема планируется
