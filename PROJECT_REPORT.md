## 📞 Инструкции для проверки

### Запуск локально:
```bash
cd shirokiy_messenger
..\shirokiy_env\Scripts\activate  # Windows
python manage.py runserver
```

### Доступ:
- **Приложение**: http://127.0.0.1:8000
- **Админ-панель**: http://127.0.0.1:8000/admin/
- **Логин админа**: admin
- **Пароль админа**: admin123

### Тестирование:
1. Зарегистрируйте 2-3 пользователей
2. Создайте чат между ними
3. Отправьте сообщения
4. Попробуйте создать группу
5. Загрузите аватарки
6. Подпишитесь на пользователей
